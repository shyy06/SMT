# SMT — 桌面股票盯盘工具

一个轻量级的 Windows 桌面股票实时盯盘工具，窗口紧凑、可自由拖动、支持鼠标穿透。

## 功能特性

- 📊 **实时股价展示** — 自动从新浪财经获取实时行情（每 5 秒刷新）
- 🖱 **自由拖动** — 窗口可拖到桌面任意位置
- 🔒 **窗口锁定** — 锁定后窗口固定在当前坐标，不可拖动
- 👻 **鼠标穿透** — 鼠标事件穿透窗口，可直接点击下层内容
- 📌 **窗口置顶** — 窗口始终位于最上层，不被其他窗口遮挡
- ➕ **多股票支持** — 右键窗口添加股票，支持上海/深圳 A 股
- 🔴🟢 **中国炒股配色** — 涨红跌绿，符合国内习惯
- 📋 **系统托盘** — 最小化到托盘，托盘右键菜单控制所有功能

## 系统要求

- Windows 10/11 x64
- .NET 8.0 SDK 或 Runtime

## 快速开始

### 方法一：使用 Visual Studio
1. 用 Visual Studio 2022 打开 `SMT.csproj`
2. 按 F5 运行

### 方法二：使用命令行
```bash
# 安装 .NET 8.0 SDK: https://dotnet.microsoft.com/download/dotnet/8.0

# 进入项目目录
cd SMT

# 构建
dotnet build -c Release

# 运行
dotnet run
```

### 方法三：使用 build.bat
双击 `build.bat` 自动构建并提示运行

## 使用说明

1. 启动程序后，桌面出现浅色背景的股票窗口
2. **右键点击窗口** → 选择「添加股票」，输入名称和代码
3. 窗口实时显示股价、涨跌幅（红色↑ = 涨，绿色↓ = 跌）
4. **右键托盘图标** → 控制窗口行为：
   - ☑ 窗口锁定 — 锁定位置
   - ☑ 鼠标穿透 — 点击穿透
   - ☑ 窗口置顶 — 始终置顶
   - 退出 — 关闭程序

## 股票代码格式

| 输入 | 自动转换 | 说明 |
|------|----------|------|
| `600519` | `sh600519` | 6 开头 → 上海 |
| `000001` | `sz000001` | 0/3 开头 → 深圳 |
| `sh600519` | `sh600519` | 已格式化，原样保留 |
